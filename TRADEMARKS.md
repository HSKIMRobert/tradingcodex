# TradingCodex Trademark Guidelines

This document explains how the TradingCodex project marks may be used. It is
not a separate source-code license. Source code and documentation licensing is
covered by `LICENSE` and `NOTICE`.

## Project Marks

The TradingCodex project marks include:

- the name "TradingCodex"
- official TradingCodex logos, icons, badges, and visual identity assets, when
  published
- names of official TradingCodex commercial offerings, hosted services, or
  verified distributions, when published

The Apache License, Version 2.0 does not grant trademark rights.

## Allowed Uses

You may use the TradingCodex name to:

- refer truthfully to the open-source project
- describe compatibility, such as "plugin for TradingCodex" or "adapter for
  TradingCodex"
- preserve copyright, license, attribution, and provenance notices
- discuss, review, benchmark, teach, or write about the project

These uses must not imply sponsorship, endorsement, official status, or a
commercial relationship with the TradingCodex project.

## Uses That Need Permission

Written permission from the project maintainer is required to:

- market a hosted or managed service as "TradingCodex"
- present a fork as the official TradingCodex product
- use TradingCodex marks in a product name, company name, domain name, app
  listing, paid marketplace listing, certification, or verification badge
- use official logos or visual identity assets in a way that suggests
  endorsement
- sell broker adapters, market-data adapters, compliance packs, or enterprise
  extensions as officially verified by TradingCodex

## Forks and Modified Distributions

Forks are welcome. If a fork makes material changes, it should use a distinct
name and clearly state that it is not the official TradingCodex distribution.

Unmodified redistribution may keep the original project name when it is clear
that the redistribution is a copy of the official open-source project and not
a separate official service or endorsed commercial offering.

## No Investment Endorsement

Use of TradingCodex marks must not suggest that the project, maintainers, or
contributors endorse any investment strategy, research conclusion, order,
broker, market-data provider, or financial outcome.
